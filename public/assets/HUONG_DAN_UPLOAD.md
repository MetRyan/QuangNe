# Thư mục Assets
Vui lòng upload các file của bạn vào các thư mục tương ứng:

- `profile/`: Ảnh đại diện của bạn (đặt tên là `avatar.jpg`)
- `projects/`: Ảnh minh họa dự án (đặt tên là `project1.jpg`, `project2.jpg`, `project3.jpg`)
- `gallery/`: Ảnh sở thích (Trekking, Rugby)
- `docs/`: File PDF (hồ sơ năng lực, CV - đặt tên là `cv.pdf`)

Sau khi upload, website sẽ tự động nhận diện các file này.
